#### 创建新的APP模块需要在此列表增加相应APP
```
INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
    'app_name',  #增加新的app
]
```

#### 修改文件时区
```
LANGUAGE_CODE = 'en-us'
TIME_ZONE = 'UTC'
```
修改为：
```
LANGUAGE_CODE = 'zh-hans'
TIME_ZONE = 'Asia/Shanghai'
```

#### 启用时区 TIME_ZONE
```
USE_TZ = True
```
